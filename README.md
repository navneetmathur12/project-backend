# 🚀 Backend Project (Node.js + Express)

## 📌 Description
This is a backend project built using **Node.js, Express, MongoDB**.  
It includes user authentication, post creation, likes, comments, and profile picture upload.

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS (Template Engine)
- JWT (Authentication)
- Multer (File Upload)
- Bcrypt (Password Hashing)

---

## ✨ Features
- 🔐 User Register & Login (JWT आधारित)
- 👤 Profile System
- 📸 Profile Picture Upload/Delete
- 📝 Create, Edit, Delete Posts
- ❤️ Like / Unlike Posts
- 💬 Comment on Posts
- 🍪 Cookie-based Authentication
